# VBA UserForms Проекты

В этом репозитории содержатся различные проекты VBA UserForms, каждый из которых предоставляет уникальный функционал для улучшения интерфейса Excel приложений.

## Таблица проектов

| Проект | Тип | Ссылка | Описание |
|--------|------|--------|-------------|
| Accordion | Public | [UserForms-Class-Acardion](https://github.com/vbatools/UserForms-Class-Accordion-VBA/) | Класс для создания аккордеонов (сворачиваемых панелей) для организации контента в компактном виде ![demo](https://github.com/vbatools/UserForms-Class-Accordion-VBA/blob/main/User_Forms.gif) |
| Modern Style | Public | [UserForms-Class-Modern-Style](https://github.com/vbatools/UserForms-Class-Modern-Style/) | Класс для стилизации элементов управления формы, позволяющий создавать современный и привлекательный пользовательский интерфейс ![demo](https://github.com/vbatools/UserForms-Class-Modern-Style/blob/main/User_Forms.gif) |
| Multi-State CheckBox | Public | [UserForms-Class-Multi-State-CheckBox-VBA](https://github.com/vbatools/UserForms-Class-Multi-State-CheckBox-VBA/) | Класс для создания многостоятельных флажков с возможностью переключения между несколькими состояниями ![demo](https://github.com/vbatools/UserForms-Class-Multi-State-CheckBox-VBA/blob/main/User_Forms.gif) |
| Slider | Public | [UserForms-Class-Slider-VBA](https://github.com/vbatools/UserForms-Class-Slider-VBA/) | Класс для создания ползунков с возможностью настройки диапазона и шага изменения значений ![demo](https://github.com/vbatools/UserForms-Class-Slider-VBA/blob/main/User_Forms.gif) |
| TextBox Masks | Public | [UserForms-Class-TextBox-Masks](https://github.com/vbatools/UserForms-Class-TextBox-Masks/) | Класс для наложения масок на текстовые поля, обеспечивающий корректный ввод данных в заданном формате ![demo](https://github.com/vbatools/UserForms-Class-TextBox-Masks/blob/main/User_Forms.png) |
| Anchors | Private | [UserForms-Class-Anchors](https://github.com/vbatools/UserForms-Class-Anchors-VBA/) | Класс для динамического прикрепления элементов управления в Excel UserForms, позволяющий элементам адаптироваться к изменению размеров формы ![demo](https://github.com/vbatools/UserForms-Class-Anchors-VBA/blob/main/User_Forms.gif) |
| Button | Private | [UserForms-Class-Button-VBA](https://github.com/vbatools/UserForms-Class-Button-VBA/) | Класс для создания улучшенных кнопок для UserForms с расширенной функциональностью и улучшенным внешним видом ![demo](https://github.com/vbatools/UserForms-Class-Button-VBA/blob/main/User_Forms.gif) |
| Calendar | Private | [UserForms-Class-Calendar](https://github.com/vbatools/UserForms-Class-Calendar-VBA/) | Класс для создания календарей с возможностью выбора даты и времени ![demo](https://github.com/vbatools/UserForms-Class-Calendar-VBA/blob/main/User_Forms.gif) |
| Spliter | Private | [UserForms-Class-Spliter-VBA](https://github.com/vbatools/UserForms-Class-Spliter-VBA/) | Библиотека для создания разделителей между элементами управления на UserForm, позволяющая динамически изменять размеры элементов ![demo](https://github.com/vbatools/UserForms-Class-Spliter-VBA/blob/main/User_Forms.gif) |
