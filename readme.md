# VBA UserForms Проекты

В этом репозитории содержатся различные проекты VBA UserForms, каждый из которых предоставляет уникальный функционал для улучшения интерфейса Excel приложений.

## Таблица проектов

| Проект | Ссылка | Описание |
|--------|--------|----------|
| Modern Style | [UserForms-Class-Modern-Style](../UserForms-Class-Modern-Style/) | Класс для стилизации элементов управления формы, позволяющий создавать современный и привлекательный пользовательский интерфейс @User_Forms.gif |
| Multi-State CheckBox | [UserForms-Class-Multi-State-CheckBox-VBA](../UserForms-Class-Multi-State-CheckBox-VBA/) | Класс для создания многосостоятельных флажков с возможностью переключения между несколькими состояниями @User_Forms.gif |
| Slider | [UserForms-Class-Slider-VBA](../UserForms-Class-Slider-VBA/) | Класс для создания ползунков с возможностью настройки диапазона и шага изменения значений @User_Forms.gif |
| TextBox Masks | [UserForms-Class-TextBox-Masks](../UserForms-Class-TextBox-Masks/) | Класс для наложения масок на текстовые поля, обеспечивающий корректный ввод данных в заданном формате @User_Forms.gif |
| Calendar | [UserForms-Class-Calendar](https://github.com/vbatools/UserForms-Class-Calendar-VBA/) | Класс для создания календарей с возможностью выбора даты и времени ![demo](https://github.com/vbatools/UserForms-Class-Calendar-VBA/blob/main/User_Forms.gif) |
| Accordion | [UserForms-Class-Acardion](../UserForms-Class-Acardion/) | Класс для создания аккордеонов (сворачиваемых панелей) для организации контента в компактном виде @User_Forms.gif |
